# git 简介
目前世界上最先进的版本控制系统
## 创建版本库
ls -ah 可以看见隐藏的命令
初始化 git repo：git init
添加文件到 git repo：1. git add < file > 2. git commit -m < message >
# 时光穿梭机
掌握工作区的状态：git status
查看修改内容：git diff 退出（q）
